1. Compile

javac -d target/ ./ImagesToChar/src/java/edu/school21/printer/app/Main.java ./ImagesToChar/src/java/edu/school21/printer/logic/PrintImage.java

2. Usage

java -cp target Main  <1> <2> <Image path>
<1> is a character for white color
<2> is a character for black color
<Image path> is a Path to image *.bmp
